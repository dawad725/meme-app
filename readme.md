## Installation

under root directory
`yarn install`
`yarn dev:start`
to start the server

under `/client`
`yarn install`
`yarn start`
to start your localhost

## Contribution

2. clone your repo
3. create a feature branch on your repo
4. work on your issues
5. test your changes
6. push your remote branch
7. Go to your repo on Github and click `Compare & pull request` button
